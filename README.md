# Data_Visualization
About the basic functions of the Seaborn and differnt plots:

What is Seaborn?
Seaborn is a statistical data visualization library built on top of Matplotlib.

Why Seaborn?
Prettier visuals
Built-in themes
Works well with Pandas
Fewer lines of code
Best for statistical plots
